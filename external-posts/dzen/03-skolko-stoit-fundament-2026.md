## КП №1 — «Эконом», 480 000 ₽ за плиту 10×10

| Позиция | Указано в КП |
|---|---|
| Бетон | «бетон, около 30 м³» |
| Арматура | «арматура, около 1,3 тонны» |
| Опалубка | «опалубка из доски» |
| Подушка | «подушка ПГС 10 см» |
| Работа | «заливка под ключ» |
| Гарантия | устная, «5 лет» |
| Аванс | 60% «на материалы» |
| Срок | 7 дней |
| **Итого** | **480 000 ₽** |

**Что не так:**

- **Не указана марка бетона.** Это значит — М200 или хуже. Завод М300 стоит ~5800 ₽/м³, М200 — ~4400 ₽/м³. Разница на 30 м³ = 42 тысячи в карман.
- **Не указан класс арматуры.** Скорее всего А240 или А400 (старье). А500С с шагом 200 = должно быть 1,8 тонны, а не 1,3.
- **Подушка 10 см** — это для дачи 6×6 максимум, не для дома. Через 5 лет грунт «продавит» — трещина.
- **Опалубка из доски однократная** — экономия 25 тыс., но даёт неровные кромки, которые потом приходится «выравнивать раствором».
- **Аванс 60%** — это бригада без оборотных средств. Деньги уходят, бригада уезжает. Распространённая схема в Питере.

**Реальная себестоимость работ:** ~280 тысяч. Бригада зарабатывает 200к чистых.
**Скрытые расходы заказчика через 3 года:** 800-1500к на ремонт.

## КП №2 — «Средний», 720 000 ₽ за плиту 10×10

| Позиция | Указано в КП |
|---|---|
| Бетон | «М300 завод ЛСР, 28 м³» |
| Арматура | «А500С Ø12, около 1,8 т» |
| Опалубка | «щитовая многоразовая» |
| Подушка | «ПГС 20 см с трамбовкой» |
| Работа | «армирование + заливка + виброуплотнение» |
| Гарантия | 5 лет договор |
| Аванс | 30% по чекам завода |
| Срок | 14 дней |
| **Итого** | **720 000 ₽** |

**Что нормально:**
- Бетон, арматура — правильно
- Опалубка многоразовая = ровная плита
- Подушка 20 см — приемлемо для большинства грунтов
- Аванс 30% — нормальная практика
- Срок реалистичный

**Чего не хватает:**
- Геотекстиль под подушкой не указан (10к разница в смете, +5 лет к сроку службы)
- ЭППС не указан — для пучинистых грунтов важно
- Толщина плиты не указана (нужно 250-300 мм под газобетон)

Если подрядчик ответит на эти вопросы устно правильно — это нормальный вариант. Просим явно прописать в договор.

## КП №3 — «Премиум», 1 050 000 ₽ за плиту 10×10

| Позиция | Указано в КП |
|---|---|
| Бетон | «М400 W8 F200, 32 м³» |
| Арматура | «А500С Ø14, 2,4 т, двойная сетка шаг 150» |
| Опалубка | «алюминиевая многоразовая» |
| Подушка | «ПГС 40 см, геотекстиль, виброплита» |
| ЭППС | «100 мм по периметру + под плитой» |
| Гидроизоляция | «рулонная двухслойная» |
| Работа | «полный комплекс с проектом» |
| Гарантия | 10 лет |
| Аванс | 20% |
| Срок | 21 день |
| **Итого** | **1 050 000 ₽** |

**Что не так с премиумом:**
- **М400 — это перебор для частного дома.** М300 W6 F150 более чем достаточно. М400 — для промышленных полов с большой нагрузкой. Это удорожание на 70 тыс. без эффекта.
- **Арматура Ø14 шагом 150** — тоже перебор. Под газобетон достаточно Ø12 шагом 200.
- **ЭППС под плитой** — нужно ТОЛЬКО если у вас УШП (утеплённая шведская плита, с тёплыми полами). Без УШП — это маркетинг, не польза.
- **Алюминиевая опалубка** — для крупных стройплощадок, не для частного дома. Не даёт никакого качественного эффекта.

Это типичный «премиум-маркетинг»: лишние +200-300к в смету за вещи которые не нужны.
